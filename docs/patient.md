# 患者 / 家属使用指南

本系统基于《西氏内科学精要》，为患者家属提供内科常见病的通俗解释和日常指导。

> **急症请立即拨打 120，本系统不替代医生。**

---

## 可以问什么

覆盖 17 个内科专科，适合这类问题：

- "我爸有高血压，平时饮食要注意什么？"
- "妈妈 2 型糖尿病，血糖控制目标是多少？"
- "心衰患者能做什么运动？"
- "骨质疏松平时需要补什么？"
- "慢性肾病患者饮食有哪些限制？"
- "肺炎出院后怎么护理？"

专科覆盖：心血管 / 内分泌 / 呼吸 / 消化（含肝）/ 肾 / 血液 / 感染 / 风湿 / 神经 / 肿瘤 / 骨矿物代谢 / 老年 / 姑息 / 围手术期 / 男性健康 / 女性健康 / 物质依赖

---

## 回答格式

每条回答固定五段：

| 段落 | 内容 |
|------|------|
| **这是什么** | 疾病简介（1-2 句通俗解释） |
| **日常怎么做** | 生活方式、饮食、运动建议 |
| **何时就医** | 需要立即看医生的信号 |
| **常见误区** | 患者和家属最容易误解的地方 |
| **依据** | 知识来源（教材章节/页码） |

---

## 系统不回答的问题

以下类型问题系统会礼貌拒答，请直接就医或咨询专科医生：

| 类型 | 示例 |
|------|------|
| 手术 / 介入决策 | "要不要放支架？" / "要不要做搭桥？" |
| 化疗具体方案 | "应该用什么化疗药？" |
| 读化验单 / 确诊 | "帮我看化验单" / "这是什么病？" |
| 调整用药剂量 | "胰岛素能自己减量吗？" |
| 无关任务 | 写作、翻译、菜谱、天气等 |

---

## 安全声明

- **急症**（胸痛、呼吸困难、突发偏瘫、意识丧失、大出血）→ 立即拨打 **120**
- 本系统仅提供内科知识科普，所有用药方案调整须就诊
- 药物剂量、诊断、处方类问题请咨询专科医生

---

## 运行方法（技术人员）

```bash
# 第一次使用
cp .env.example .env          # 填入 DEEPSEEK_API_KEY
pip install -r requirements.txt

# 提问
./bin/ask.sh "我爸有高血压，平时饮食要注意什么？"
./bin/ask.sh "妈妈 2 型糖尿病，血糖控制目标是多少？"
./bin/ask.sh --debug "心衰患者能做什么运动？"   # 显示路由信息
```
